# Task-Tracker-Web-App
A simple, fast task management web app built with HTML, CSS, and JavaScript.
Everything runs in the browser and uses LocalStorage for persistence, so no backend required.

#Features

Add, edit, delete tasks

Mark tasks as complete/incomplete

LocalStorage persistence (tasks stay even after refresh)

Priority levels: Low, Medium, High

Due date support

Search, sorting, and filtering

Clean and responsive UI

No frameworks, no dependencies

#Tech Stack

HTML5

CSS3

JavaScript (ES6)

Browser LocalStorage

#Project Structure
Task-Tracker-Web-App/
│── index.html
│── style.css
│── script.js

#How to Run

Just open index.html in any modern browser.
No server needed.

#How to Use

Enter task title

(Optional) Add description, due date, and priority

Click Save Task

Use filters and search to manage your tasks

Tasks are saved automatically in your browser

Deployment

You can deploy it using GitHub Pages:

Go to Settings → Pages

Select:

Branch: main

Folder: /root

Save

Your live link will be generated automatically
